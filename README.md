## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript backend repository for technical test.

#### Technologies Used

- **Language**: Typescript
- **Framework**: NestJS.
- **Database**: PostgreSQL 16 & TypeORM.
- **Documentation**: Swagger.
- **User management**: Password hashing with BcryptJS & JsonWebToken as token manager.

## Getting started

First setup the env file:

```bash
cp .env.example .env
```

and add the correct values for the variables.

Then install the dependencies:

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test:unit

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Deploy

```bash
# Install vercel globally
$ npm i -g vercel

# Login with vercel
$ vercel login

# Build the app
$ npm run build

# Deploy to prod
$ vercel --prod
```
